# 📚 AI Study Helper (Android)

AI Study Helper is a simple Android prototype application that helps students quickly understand academic concepts and summarize study material using Generative AI.

The app allows users to ask academic questions or paste study notes and receive simplified explanations or concise summaries generated by an AI model.

This project was built as part of a **Generative AI Android Development internship task** to demonstrate the integration of AI services into a mobile application.

---

# 🚀 Features

### 1️⃣ Question & Answer (Concept Explanation)

Users can type a study-related question and receive a simple explanation from the AI.

Example:

Input:  
What is Newton's Second Law of Motion?

Output:  
A simplified explanation with an easy example.

---

### 2️⃣ Text Summarization

Users can paste study notes or paragraphs and receive a short summary highlighting the most important points.

Example:

Input:  
Long paragraph from textbook

Output:  
3–5 bullet points summarizing the key ideas.

# 🧠 AI Model

The application uses the Gemini API for generating responses.

Current model used:

gemini-2.5-flash-lite

This model provides fast responses suitable for educational Q&A and summarization tasks.

---

# 📱 Application Interface

The app includes a simple and clean UI with:

- Text input field
- Mode selection (Q&A / Summarize)
- Submit button
- AI response display card

---

# ⚙️ How It Works

1. User enters a question or text
2. The app creates a prompt based on the selected mode
3. The prompt is sent to the Gemini API
4. The AI response is received and displayed in the app
